## 💾 Kurulum

1. **[CloudStream Releases](https://github.com/recloudstream/cloudstream/releases)** adresinden güncel APK dosyasını indirip kurun.
2. Uygulamanın yüklü olduğu cihazda depoyu otomatik yüklemek için **[Buraya Tıklayın](cloudstreamrepo://raw.githubusercontent.com/ilkelkullanici/ilkel-cloudstream/refs/heads/builds/repo.json)**.
